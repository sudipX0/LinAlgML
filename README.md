# LinAlgML
Linear Algebra for Machine Learning

## Overview
This repository explores eigenvalues and eigenvectors through linear transformations and their practical applications in machine learning and web analytics.

## Contents
- **Eigenvalues and Eigenvectors Analysis**: Mathematical foundations, definitions, and computational methods using Python
- **Standard Transformations**: Analysis of reflection, shear, rotation, scaling, and projection transformations in 2D space
- **Web Navigation Model**: Application of eigenvalues to model webpage browsing patterns using Markov matrices and discrete dynamical systems

## Key Topics
- Computing eigenvalues and eigenvectors with NumPy
- Visualizing linear transformations
- Markov matrices and stochastic processes
- Steady-state analysis for web navigation

## Technologies
- Python 3
- NumPy
- Matplotlib
